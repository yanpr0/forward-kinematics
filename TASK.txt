Вам необходимо реализовать три функции, которые расчитывают прямую кинематику скелета персонажа.

1) есть Makefile, который теоретически должен работать везде, если есть проблемы с построением билда, можно попробовать собрать visual studio.
После удачного запуска вы должны увидеть аналогичную картинку с step1.mp4

2) реализуйте forward_kinematics_full
В данной функции нужно заполнить два массива с мировыми трасформациями(смещением и вращением), имея локлаьные трансформации,
а также индекс родительской ноды.

При успешной реализации будет получен дергающийся, но уже адекватно анимирующийся человечек см step2.mp4

3) forward_kinematics, forward_kinematics_velocity
Данные функции используются в алгоритме motion matching, который управляет выбором анимаций.
В данных функциях нужно вычислить мировые параметры только для одной ноды(параметр - int bone).

После правильной реализации этих функций получите финальный результат step3.mp4
